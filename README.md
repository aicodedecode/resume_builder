# Instant Game Pack (No Sign-In)

A ready-to-use, fully client-side pack of **10 browser games** that work without accounts or backend services.

## Included games

1. Tic-Tac-Toe
2. Rock Paper Scissors
3. Hangman
4. Memory Match
5. Number Guess
6. Whac-a-Mole
7. Snake
8. 2048 (4x4)
9. Minesweeper (6x6)
10. Mini Sudoku (4x4)

## Run locally

Open `index.html` directly in any modern browser.
